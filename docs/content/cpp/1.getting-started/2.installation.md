## Requirements

- **C++17** or later
- **[oneTBB](https://github.com/oneapi-src/oneTBB)**

### Installing TBB

::code-group
```bash [macOS]
brew install tbb
```
```bash [Linux]
sudo apt-get install libtbb-dev
```
```powershell [Windows]
vcpkg install tbb
```
::

::tip{icon="i-lucide-settings"}
If TBB is in a non-standard location:
```bash
cmake -B build -DTBB_ROOT=/path/to/tbb
```
::

## pip install

The easiest way to get trueform C++ headers is via pip. This installs both [Python bindings](/py/getting-started) and C++ headers:

```bash
pip install trueform
```

### CMake Integration

Use the header-only library as a CMake package:

```cmake [CMakeLists.txt]
cmake_minimum_required(VERSION 3.16)
project(my_app LANGUAGES CXX)

find_package(trueform REQUIRED CONFIG)

add_executable(my_app main.cpp)
target_link_libraries(my_app PRIVATE tf::trueform)
```

Configure your build so CMake can find the package installed by pip:

```bash
cmake -B build -Dtrueform_ROOT=$(python -m trueform.cmake)
cmake --build build
```

::tip{icon="i-lucide-info"}
You can also query paths directly:
- `python -m trueform.cmake` — CMake config directory
- `python -m trueform.cmake --include_dir` — C++ headers path
- `python -m trueform.cmake --cmake_dir` — Same as default
::

## System Install

Clone and install trueform system-wide:

```bash
git clone https://github.com/polydera/trueform.git
cd trueform

cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
cmake --install build --config Release
```

Then use `find_package` in your project:

```cmake [CMakeLists.txt]
cmake_minimum_required(VERSION 3.16)
project(my_app LANGUAGES CXX)

find_package(trueform REQUIRED CONFIG)

add_executable(my_app main.cpp)
target_link_libraries(my_app PRIVATE tf::trueform)
```

```bash
cmake -B build
cmake --build build
```

::tip{icon="i-lucide-folder"}
To install to a custom location:
```bash
cmake --install build --prefix ~/mylibs
```
Then configure with:
```bash
cmake -B build -DCMAKE_PREFIX_PATH=~/mylibs
```
::

## CMake FetchContent

For a self-contained project without system installation:

```cmake [CMakeLists.txt]
cmake_minimum_required(VERSION 3.16)
project(my_app LANGUAGES CXX)

include(FetchContent)

FetchContent_Declare(
  trueform
  GIT_REPOSITORY https://github.com/polydera/trueform.git
  GIT_TAG        main
)

FetchContent_MakeAvailable(trueform)

add_executable(my_app main.cpp)
target_link_libraries(my_app PRIVATE tf::trueform)
```

```bash
cmake -B build
cmake --build build
```

::tip{icon="i-lucide-git-branch"}
Pin to a specific version using a git tag or commit hash in `GIT_TAG`.
::

## Conan

Create the package from the recipe included in the repository:

```bash
git clone https://github.com/polydera/trueform.git
conan create trueform/conan/trueform --build=missing
```

Then add to your `conanfile.txt`:

```ini [conanfile.txt]
[requires]
trueform/[>=0.1.0]
```

Or in `conanfile.py`:

```python [conanfile.py]
def requirements(self):
    self.requires("trueform/[>=0.1.0]")
```

Then generate and build:

```bash
conan install . --build=missing
cmake --preset conan-release
cmake --build --preset conan-release
```

::tip{icon="i-lucide-package"}
The recipe automatically pulls TBB as a dependency. If your Conan profile defaults to an older C++ standard, add `-s compiler.cppstd=17` to the commands.
::

```cpp [main.cpp]
#include <trueform/trueform.hpp>
#include <iostream>

int main() {
    auto pt = tf::make_point(1.0, 2.0, 3.0);
    std::cout << "trueform is working!" << std::endl;
    return 0;
}
```
